> **Instrutor:** Manu Rossatt  
> *Conteúdo focado em Engenharia, Arquitetura de Sistemas e Gestão Ágil.*

Este repositório centraliza os arquivos e aulas desenvolvidos para capacitação técnica, cobrindo desde a base do hardware até as metodologias de gestão de software.

---

## 📂 Visão Geral dos Módulos

### 1. 🌐 Arquitetura de Redes IoT
Focado na integração entre hardware e software. Explora a comunicação entre dispositivos de borda e sistemas de processamento de dados.
*   **Tecnologias:** C++ (Arduino), Python.
*   **Destaque:** Fluxo de dados entre camadas de percepção e aplicação via comunicação Serial/MQTT.

### 2. 📋 Engenharia de Requisitos
O guia completo do levantamento à documentação. Focado em transformar a dor do cliente em especificações técnicas precisas.
*   **Metodologias:** Scrum, Kanban, User Stories.
*   **Técnicas:** Entrevistas passo a passo, Brainstorming, Questionários e Prototipagem (Figma/Papel).
*   **Documentação:** Diferenciação de Requisitos Funcionais e Não Funcionais.

### 3. 🖥️ Sistemas Operacionais (SO)
Uma análise comparativa sobre a gestão de recursos de hardware e os principais ecossistemas do mercado.
*   **Sistemas:** Windows (NT), Linux (Kernel Open Source), iOS (Unix/XNU).
*   **Conceitos:** Gerenciamento de processos, memória virtual, sistemas de arquivos e virtualização (Docker/VM).

---

## 🛠️ Portfólio de Projetos Python (Destaque)
Além dos materiais teóricos, este repositório inclui o **"Simulador de Robô Industrial"**, um projeto prático que demonstra:
*   **Lógica Sólida:** Controle de passos e ângulos em Python.
*   **Resiliência:** Resolução de 10 grandes desafios de *debugging* (O Caça-Erros).
*   **Automação:** Scripts utilitários para cálculos do dia a dia.
